# Destiny Clothing

Premium streetwear storefront prototype for **Destiny Clothing — Your Path. Your Rules.**

## Publish with Vercel + GitHub

1. Create a new, empty GitHub repository named `destiny-clothing`.
2. Open a PowerShell window in this folder and run:

   ```powershell
   git init
   git add .
   git commit -m "Initial Destiny Clothing website"
   git branch -M main
   git remote add origin https://github.com/YOUR-USERNAME/destiny-clothing.git
   git push -u origin main
   ```

3. Sign into [Vercel](https://vercel.com), select **Add New → Project**, then import the GitHub repository.
4. Leave the framework preset as **Other** and deploy. No build command is needed for this static site.

Every push to `main` will update the live website automatically.

## Local preview

Open `index.html` directly in a browser, or run:

```powershell
npx serve .
```

## Before taking real orders

This is a front-end prototype. Add a product database, secure checkout/payment provider, authentication, privacy policy, and legal/business details before accepting orders.
